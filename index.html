项目演示

A Stereo-Lidar SLAM System

我们搭建了一套基于激光雷达和双目视觉的SLAM（同时定位与建图）系统，该系统可以实现室内外场景的高精度三维重建。

海报 / 演示视频: youku youtube

3D Object Tracking for Augmented Reality

基于三维物体跟踪的增强现实应用演示。

演示视频: youku youtube

3D Paint

三维物体位姿跟踪技术使得我们可以在三维物体的表面进行绘画，并从任意角度欣赏三维绘画作品。

演示视频: youku youtube

Fast 3D Object Tracking with Edge Distance Fields

基于边缘距离场的三维物体位姿跟踪，该算法可仅依靠约100个边缘点来快速跟踪物体的六自由度位姿，并能检测遮挡。

演示视频: youku youtube

A Vision-aided Robot Arm

我们搭建了一套视觉辅助的机械臂装置，并展示了一些可能的应用示例，例如物品自动抓取和分拣、激光雕刻、三维打印、平面目标跟踪、空中加油仿真平台等。

演示视频: youku youtube
